## ANTES vs DEPOIS

### Botão
```
❌ ANTES: <button class="btn btn-primary">Ação</button>
✅ DEPOIS: <button class="btn-modern btn-primary">Ação</button>
```

### Card
```
❌ ANTES:
<div class="card shadow-sm">
  <div class="card-body">...</div>
</div>

✅ DEPOIS:
<div class="card-modern card-accent">
  <div class="card-header-modern"><h3>Título</h3></div>
  <div class="card-body-modern">...</div>
</div>
```

### Tabela
```
❌ ANTES: <table class="table table-hover">
✅ DEPOIS: <table class="table-modern">
```
